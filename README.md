# 初中竞赛衔接·奥数训练（六年级）— v6-static（无 server.py）

这一版专门解决你反馈的：
- ✅ **不依赖 server.py**（纯静态网页即可用）
- ✅ **组卷打印**：勾选题目 → 打开打印版 → 浏览器打印/另存为 PDF；同时可导出 Word（.doc）
- ✅ **错题本**：自动收集做错且未掌握的题；支持一键导出错题练习卷（PDF/Word）
- ✅ **拍照批改（本机）**：上传打印卷照片 → 识别二维码得到“做了哪些题” → 手动录入答案 → 自动判定并写入学习记录/错题本  
  > 说明：手写答案 OCR 在纯前端环境下很难做到稳定准确，因此本版默认“题目自动识别 + 手动录入答案”。你如果确实需要 OCR，我可以再给你做一个“需要联网加载 OCR 模型”的增强版。

题库方面仍保持：
- ✅ 题库 **1050 题（>1000）**
- ✅ 题目按【知识点 topic】+【奥赛题型/模型 model】组织
- ✅ 每题型（模型）都有「题型讲解 + 例题讲解」
- ✅ 变式追击：同题型/模型一次给出 **12 题（≥10）**

---

## 运行方式
### 方法 A：直接打开（离线可用）
解压后双击 `index.html` 即可运行。  
> 若浏览器限制 `file://` 读取 JSON，本项目已提供 `assets/data.js` 离线兜底。

### 方法 B：本地小服务器（更推荐，更稳定）
在解压目录打开终端：
```bash
python3 -m http.server 8000
```
浏览器访问：
- http://localhost:8000

---

## 功能入口
网页顶部菜单：
- **组卷打印**：勾选题目 → “打开打印版（PDF）” → 打印对话框里选择“另存为 PDF”
- **错题本**：自动收集错题 → 一键导出错题卷
- **拍照批改**：上传你打印出的卷子照片（每题左侧有二维码）→ 识别题目 → 录入答案 → 判定并记录

---

## 数据保存在哪里？
- 学习记录、错题本、组卷选题等都保存在浏览器本机 `localStorage`（同一台电脑/同一浏览器内有效）。
- 在「画像」页面仍可导出 `learning_record.json` 备份。
